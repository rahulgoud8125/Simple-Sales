# Simple Sales Dashboard (Elevate Labs Internship)
**Objective**

Build an interactive dashboard to analyze sales by month, region, and category using the Superstore dataset.

**Dataset Used**

Superstore (1).csv 


**Steps Performed**

Imported the dataset into Power BI

Cleaned missing values and formatted columns

Converted Order Date → Month–Year

**Added visuals:**

Line Chart: Sales over time

Bar Chart: Sales by Region

Donut Chart: Sales by Category

Added slicers for Region and Category

Applied color formatting for clarity

**Insights**

Sales peak in Oct–Dec

West region leads in overall sales

Technology category gives highest revenue

Furniture category has lowest profit margins
